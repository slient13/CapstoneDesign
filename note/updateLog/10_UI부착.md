## 21-10-13 : 장비 UI 부착.
기능 : 
- 최대 4개까지 장비 장착 및 장착 현황 확인.
- 장착된 슬롯 클릭 시 별도 툴팁창 열림.
- 툴팁창에서 esc 를 누르거나 창 밖을 누르면 사라짐
- 툴팁창에서 `장착 해제`를 누르면 장비가 해제됨. 해제된 장비는 인벤토리로
- 하단에 일부 스탯 현황을 볼 수 있음.

툴팁 창 : 
- 아이템의 이름과 설명을 볼 수 있음.
- 아이템 설명은 현재 스탯 변화량으로만 표기됨.
