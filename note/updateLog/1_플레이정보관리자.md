## 21-05-21 : 기능 개선, 변경점.
변경점
- `min, max` 속성 추가.
    - int 형 혹은 float 형 정보에서 작동.
    - 값 변경 시 최소, 최대값을 벗어나지 못하게 함.
    - 생략 시 최소 -999999, 최대 2000000000 로 자동 지정.
- 입력 양식 변경.
    - 아이템 정보와 유사하게 `code, type, value, min, max, end` 형태로 입력 가능.
    - `$mode = value` 형태.
    - `short = $code` 형태로 기술하면 이전처럼 한 줄 짜리 약식으로도 입력 가능.

기능 개선
- 위 변경점에 관련된 기능 추가.
